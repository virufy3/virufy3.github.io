AIテクノロジーによるCOVID-19の早期検出

私たちと共にCOVID-19と闘いましょう！

私たちのミッション

私たちのビジョンは、世界中の誰もがCOVID-19(新型コロナウイルス)の感染を数秒で診断できる スマートフォンアプリを開発することです。 最先端の人口知能技術を使用する事で、COVID-19によって損傷された肺の特有な呼吸パターンを 識別することができます。 この特有な呼吸パターンは人間の耳では聞き取れず、人工知能技術（AI）にしか聞き取れません。 私たちの広範囲にわたる調査から、75％以上の精度を出し、 このパンデミックの抑制を支援できると信じています。

私たちについて

私たちはAIエンジニア、モバイルWebアプリ開発者、医療専門家をはじめ、 若者が中心になっている非営利団体です。 私たちは現在の世界の状況を救うために最善を尽くしたいと思っており、 世界の公衆衛生・健康にとって極めて重要な時期に変化をもたらす機会に感謝しています。 私たちのチームは スタンフォード大学のCOVID-19 response Innovation Lab (新型コロナ対応の研究ラボ） の一員であり、 One Young World にも紹介されています。あなたの協力は１９０カ国以上にわたる何億人もの命のためになります！
